# 技术栈（锚定项）

## 后端

| 项 | 选型 |
|----|------|
| 语言 | Java 17 |
| 框架 | Spring Boot 3.2.5 |
| 模块 | `tt-common` / `tt-application` / `tt-infrastructure` / `tt-admin-api` |
| 持久化 | MySQL 8 + JdbcTemplate + MyBatis-Plus（用户/审计） |
| 迁移 | **Flyway**（`db/migration/V*.sql`） |
| 权威 DDL | **`db/schema.sql`** |
| 安全 | Spring Security + JWT |
| 消息 | RabbitMQ（外部同步可选） |
| 外部同步 | Jsoup / RSS 解析；`tt.sync.mode=direct\|queue`；`@Scheduled` 按 `interval_minutes` 自动触发 |

## 前端

| 项 | 选型 |
|----|------|
| 框架 | Vue 2.7 + vue-router 3 |
| 官网 | Vant 2 |
| 管理端 | Element UI 2 |
| 构建 | Vite 2.9 |
| 公共请求 | `client/shared/request.js` |

## 部署

| 项 | 选型 |
|----|------|
| 容器 | Docker Compose 3.8 |
| 服务 | mysql · rabbitmq · **tt-admin-api** · nginx(web) |
| 环境变量 | `DB_*` · `RABBITMQ_*` · `TT_JWT_SECRET` · `SPRING_PROFILES_ACTIVE=prod` |
| CI | GitHub Actions `ci.yml` |

## 端口

| 服务 | 端口 | 说明 |
|------|------|------|
| tt-admin-api | 8096 | 应用 HTTP（与 IMP 8080 错开） |
| portal dev | 5173 | Vite |
| admin dev | 5174 | Vite |
| HTTP (compose) | **8888** | Nginx 官网+管理端（与 IMP 无冲突） |
| MySQL (compose 宿主机) | **3307** | 映射容器 3306（IMP 用 3306） |
| RabbitMQ (compose 宿主机) | **5673** / **15673** | 管理台（IMP 用 5672/15672） |
