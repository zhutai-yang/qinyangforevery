# 00 - 切片概述

## 目标

完成可运行的 **服务端骨架**、**最小表结构**、**管理员登录**，为后续后台 CRUD 与 public API 打基础。

## 方案对照

| 编号 | 内容 |
|------|------|
| S01 | `/`、`/admin`、`/api`、`/api/health` |
| S02 | `fnd_*`、`evt_event`、`evt_venue`、`sch_*`、`reg_athlete`、`rec_result` 等最小集 |
| S03 | `POST /api/admin/auth/login`、`requireAdmin` 中间件 |

## 依赖

无前置切片；可与 **切片UI-前端基础设施** 并行。
