# 01 - 工程初始化（S01）

## 实现要点

- **Spring Boot 3.2.5**：Maven 多模块，`tt-admin/tt-admin-api`，入口 `TtAdminApplication`。
- **模块**：`tt-shared`（common / application / infrastructure）+ `tt-admin-api`（Controller）。
- **健康检查**：`GET /api/health`（探测数据库 `SELECT 1`）。
- **API 前缀**：`/api/public/**`（公开）、`/api/admin/**`（JWT，登录除外）。
