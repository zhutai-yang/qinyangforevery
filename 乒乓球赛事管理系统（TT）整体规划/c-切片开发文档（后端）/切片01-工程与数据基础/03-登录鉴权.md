# 03 - 登录鉴权（S03）

## API

- `POST /api/admin/auth/login` → 返回 `token`（JWT）
- `POST /api/admin/auth/logout`
- `GET /api/admin/auth/me`（Header：`Authorization: Bearer <token>`）

## 服务端

- Spring Security：`/api/admin/**` 需认证（`/api/admin/auth/login` 除外）。
- 密码 **BCrypt**；登录限流用 **MySQL** 表 `sys_login_rate`（`LoginRateLimiter`），不依赖 Redis。
