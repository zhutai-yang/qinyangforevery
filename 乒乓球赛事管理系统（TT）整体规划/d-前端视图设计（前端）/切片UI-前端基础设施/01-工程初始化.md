# 01 - 工程初始化

## 步骤

1. Node **18+**（与根目录 `package.json` engines 一致）。
2. 目录 **`client/`**：`shared/`、`portal/`、`admin/`；Vite 见 `client/vite.*.config.js`。
3. 安装：`npm install --legacy-peer-deps`

## 检查项

- [ ] `npm run build:all` 无报错
- [ ] Java 后端已启动时，`dev:portal` / `dev:admin` 能访问 `/api/health`
