# 00 - 切片概述（前端基础设施）

## 范围

- Vue 2.7 双入口：**官网**（`client/portal` + **Vant 2**）、**管理后台**（`client/admin` + **Element UI 2**）。
- 公共请求层：`client/shared/request.js`（axios 封装）。
- 开发时 Vite 代理 `/api/` → `tt-admin-api`（**8096**）；管理端 **JWT**。

## 依赖

- **后端**：`tt-admin-api` 提供 `/api/*`。
- UI 可在 mock 下先行开发。

## 产出物

- `npm run build:all` → `dist/portal`、`dist/admin`
- 生产：Docker Compose 中 **nginx(web)** 托管静态资源，反代 `/api/` → `tt-admin-api`
