# design-sync（TT 版）

## 何时使用

| 场景 | 紧急度 |
|------|--------|
| baseline 与 `02-技术栈.md` 冲突 | 🔴 阻断 |
| API 与现有公开契约不一致 | 🔴 阻断 |
| 涌现需扩展顶层定义 | 🟡 切片完成前 |

## 产出路径

```
c-切片开发文档（后端）/切片N-*/design-sync-proposals/DS-{N}-{01}.md
```

## 提案模板

```markdown
# DS-{N}-01: 标题
- **类型**: 设计缺口 / 规则冲突 / 涌现升级
- **目标文档**: `a-锚定项/02-技术栈.md` 或 切片概述
- **现状**: ...
- **建议变更**: ...
- **影响切片**: ...
- **状态**: 📝待决策 / ✅接受 / ❌拒绝
```

## 决策后

- ✅ 接受 → 更新顶层文档 → 重新 `/slice-bootstrap`
- ❌ 拒绝 → baseline 按现顶层收敛
- 禁止「一期暂不实现」不写规则 — 写完整规则并 `@since` 标注实现时机
